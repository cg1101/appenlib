Network Wireless Electronic Library

APXLIB is the Network Wireless on-line electronic document library system. The name APXLIB derives from "AUTOPLEX Library"--the name of the 1st generation Lucent wireless system.   This electronic document library system  has been on-line since 1988 and contains thousands of  documents. You will need  WEB ID and password in order to search and view/print documents.   It is easy to obtain a WEB ID via the NWS WEB ID link.
